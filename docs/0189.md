* 三次翻转
```cpp
1234 567 => 目标567 1234
765 4321
567 1234
```
* 实现如下
```cpp
class Solution {
public:
    void rotate(vector<int>& nums, int k) {
        k %= size(nums); // k可能超出数组长度
        reverse(begin(nums), end(nums));
        reverse(begin(nums), begin(nums) + k);
        reverse(begin(nums) + k, end(nums));
    }
};
```
