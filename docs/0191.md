* 对一个数和比它小1的数做与运算，即可消除这个数二进制最低位的1
```cpp
最低位是1，则等价于把最低位的1改为0
1111 & 1110 = 1110
1011 & 1010 = 1010
xxx1 & xxx0 = xxx0

最低位不是1，则找到最低位的1，比它小1的数该位为0，之后为1
1110 & 1001 = 1100
1100 & 1011 = 1000
1000 & 0111 = 0000
xxx100000...（n个0） & xxx011111...（n个1） = xxx000000（n+1个0）
```
* 逐次消除最低位的1直到值为0，消除的次数即为结果
```cpp
class Solution {
public:
    int hammingWeight(uint32_t n) {
        int res = 0;
        while (n)
        {
            ++res;
            n &= n - 1;
        }
        return res;
    }
};
```
