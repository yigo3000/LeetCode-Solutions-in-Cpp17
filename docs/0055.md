* 记录每个位置可以到达的最大右边界，如果当前位置超出最大右边界，则无法跳到当前位置，也就无法到达终点（如果能到达终点，则终点前的每个位置都应该可到达）
```cpp
class Solution {
public:
    bool canJump(vector<int>& nums) {
        int mx = 0;
        for (int i = 0; i < size(nums); ++i)
        {
            if (i > mx) return false;
            mx = max(mx, i + nums[i]);
        }
        return true;
    }
};
```
