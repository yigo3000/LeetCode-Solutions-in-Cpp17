* 用双指针从两侧往中间移动，如果其中出现两侧不等的情况，则不是回文串
```cpp
abcddba
|     |
l     r

abcddba
 |   |
 l   r

abcddba
  | |
  l r
不等，因此不是回文串
```
* 题目只考虑字母和数字字符，忽略大小写，对这些情况进行处理即可
```cpp
class Solution {
public:
    bool isPalindrome(string s) {
        int l = 0;
        int r = size(s) - 1;
        while (l < r)
        {
            if (!isalnum(s[l]))
            {
                ++l;
            }
            else if (!isalnum(s[r]))
            {
                --r;
            }
            else // 左右两侧都是字母和数字字符
            {
                if (tolower(s[l++]) != tolower(s[r--])) return false;
            }
        }
        return true;
    }
};
```
